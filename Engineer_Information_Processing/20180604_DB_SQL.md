### DDL
#### DDL(Data Define Language)의 개념
* 스키마, 도메인, 테이블, 뷰, 인덱스를 정의하거나 변경 또는 제거할 때 사용하는 언어
* DDL로 정의된 내용은 메타데이터가 되며, 시스템 카탈로그에 저장
* 종류  
| 명령문 | 헤더 |  
| :------------- | :------------- |  
| CREATE | 스키마,도메인,테이블,뷰,인덱스를 정의 |  
|  ALTER | 테이블에 대한 정의를 변경 |  
|  DROP  | 스키마,도메인,테이블,뷰,인덱스를 정의 |
